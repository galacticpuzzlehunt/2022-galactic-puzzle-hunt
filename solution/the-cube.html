Solution The Cube
Answer WHACK TO THE FUTURE

Written by Brian Chen, CJ Quines, Anderson Wang, and Ben Yang

Part 1 (2021)

The first part of this puzzle asks us to collect all 14 energy nodes. We are given a map, which shows us a rectangular prism, a radar that appears to show distances to the nodes, and a fuel gauge. Looking at the puzzles on 2021, we see all three main rounds (Forward, Hibernating and Flying South, and World‚Äôs End Concert) are visible at once. Furthermore, there is now a crosshair icon next to each of the puzzles.

Our first step is to understand what sort of space we are in and how to navigate. By clicking on the crosshair icons, we can observe the fuel gauge decreasing and the distances on the radar changing. Through playing around with this, we can hypothesize and confirm several facts:

  • We are inside a 8×10×12 rectangular prism
  • One unit of fuel is spent for every unit of distance traveled
  • The radar shows Manhattan distances to the energy nodes
  • Clicking on the crosshair icon next to a puzzle will cause you to move to a target value within one dimension (and keep you stationary in the other two dimensions)
  • Each of the rounds corresponds to a different dimension, and each puzzle corresponds to a different target value in that dimension
  • When we pass over an energy node it is collected and the letter on said node is revealed

From this amount of information, it is possible to devise a set of movements that will collect all energy nodes. However, we may also notice that the correspondence of puzzles to target values is given by the order in which feeder answers are used in the three main round metas. That is, if a puzzle‚Äôs answer provides the nth letter to the answer of the corresponding meta, then that puzzle‚Äôs crosshair icon will set the corresponding coordinate to n. We will adopt the convention of using the x-coordinate for the dimension of size 8, the y-coordinate for the dimension of size 10, and the z-coordinate for the dimension of size 12.

Part 2 (2022)

We are now presented with the long string that previously appeared in The Line and The Square, along with a list of trigrams. The reappearance of this long string suggests that we may once again wish to write the letters along a cyclic path. Looking back at the coordinates of the energy nodes we found in the first part, we notice that there is a unique (modulo traversing it forwards or backwards) cyclic path that collects all of them while moving directly to an uncollected energy node at each step. This path has length 77 (counting the starting location twice), which is the length of the string. Trying both traversal orders, when we write the letters of the string along the path, we notice that one of them has the letters from the nodes align with the letter from the string, which further confirms that this step is the correct thing to do. The path now gives us an order on the set of 14 letters from the energy nodes, and it spells out the cluephrase SEE DEVICE NAMES.

Easter Egg

We also have a bonus hidden in this set of metapuzzles (The Line, The Square, The Cube). The long string shared among these puzzles is used to extract a total of three cluephrases and three answers:

This uses up almost all of the letters in the string, except for 8. These 8 not yet constrained letters are at positions 3, 14, 18, 33, 35, 49, 51, and 68. If we look at the letters at these positions, we get the phrase USE PAPER, suggesting a strategy that, according to the mechanics of the game rock-paper-scissors, may work against the asteroid. Longtime GPH participants may also recognize this as a callback to our first hunt in 2017, the Puzzleball Championships.

Author’s Notes

The endgame and metameta was written after all the other metas and answers were basically finalized (except of course for the yet unconstrained letters in the string from The Line/The Square, which we always intended to use). It was purely a coincidence that all three metas extracted one letter from each feeder answer, which we were happy to be able to use in this puzzle.

With computer assistance, we tried many, many cluephrases, sets of indexes, and 3D paths to find one that was as neat as possible. About the only aspect in which the metameta affected the rest of the hunt was that we used the word ‚Äúdevice‚Äù in story text to describe the three meta answers to accommodate the cluephrase we ended up with.

At first, the endgame worked by having solvers navigate ‚Äúto‚Äù a puzzle by actually submitting the correct answer to that puzzle. Flavor-wise, we thought this would conclude the time loop theme in the classic way by requiring solvers to achieve a ‚Äúgolden loop‚Äù in how they solved the main round puzzles, similar to how they solved the intro round puzzles in a different way to access the main round.
